# Интернет магазин

Пет проект на основе интернет магазина

## Архитектура проекта

- Frontend: `Vue`
- Backend: `nest` `Микросервисы`
    - Users
    - Products
    - Orders
    - Warehouse
    - Payments `optional`
    - Notifications `optional`

## Цели

Изучить микросервисную архитектуру, DDD паттерн.
Научиться управлять набором контейнеров
